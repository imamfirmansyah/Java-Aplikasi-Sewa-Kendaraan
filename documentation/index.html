3. Menjalankan Java Client

3.1. Menggunakan IDE Netbean

Untuk memudahkan, saya menggunakan IDE Netbean untuk menjalankan java client, namun sebelum menjalankan pastikan Maven sudah terinstal dan siap dijalankan pada komputer anda.

  • Dalam dokumentasi ini saya menggunakan Windows dengan Netbean versi 8.1, silahkan menyesuaikan dengan versi netbean yang anda miliki
  • Klik File > Open Project atau Ctrl + Shift + O untuk membuka project java maven dan arahkan pada direktori folder tempat kita mengextract file repositori yang telah kita download sebelumnya
  • Jika terdapat error seperti gambar di bawah ini, berarti repositori dependency yang dipanggil pada pom.xml belum dimiliki oleh repositori maven pada komputer anda,
  • Untuk mengatasi error tersebut, jalankan saja Main.java nya maka, maven akan otomatis mendownload repositori yang dibutuhkan. Untuk menjalankannya, buka file Main.java lalu klik Run > Run File atau Shift + F6
  • Pastikan anda terkoneksi dengan internet, agar maven dapat mendownload dependency yang dibutuhkan, dan berikut adalah tampilan saat proses download repositorinya
  • Jika proses download sudah selesai maka akan muncul Form Login untuk masuk ke Aplikasi Penyewaan Kendaraan
  • Aplikasi telah siap untuk dijalankan

3.2. Menggunakan Console

Pastikan Maven sudah terinstal dan siap dijalankan pada komputer anda, lalu untuk menjalankan Maven Melalui Console pertama-tama masuk ke direktori folder tempat kita mengextract file repositori yang telah kita download sebelumnya, masuk ke folder java-client

Compile dengan mengetik : 

mvn clean package
Setelah proses download selesai, run Main.java dengan perintah 
mvn exec:java -Dexec.mainClass=com.firmansyah.imam.aplikasi.uts.Main
Tunggu hingga maven selesai mendownload repositori dependency yang dibutuhkan,

4. Menggunakan Aplikasi

Untuk menjalankan Aplikasi Penyewaan Kendaraan, ikuti langkah-langkah pada tahapan 3. Menjalankan Java Client, jika sudah muncul Form Login maka Aplikasi siap untuk dijalankan

  • Langkah Pertama yang harus kita lakukan adalah mendaftar terlebih dahulu, klik Tombol Register untuk mendaftar, lalu akan muncul Form Register seperti gambar dibawah ini :

  • Masukkan Nama, Username, dan Password untuk contoh saya menginput :
    Nama : administrator
    Username : admin
    Password : admin
    Jika Berhasil maka akan muncul notifikasi seperti ini :

  • Login dengan username dan password yang telah dibuat sebelumnya, Jika Login Berhasil maka akan muncul notifikasi seperti ini :

  • Berikut ini adalah tampilan utama Aplikasi Penyewaan Kendaraan

  • Ada 3 bagian utama dalam Form Sewa

    Area yang Berwarna Biru merupakan bagian untuk menyimpan transaksi penyewaan, ID Pelanggan merupakan data yang otomatis dimunculkan sesuai dengan data pelanggan yang akan kita inputkan pada Form Pelanggan, begitu juga dengan ID Kendaraan yang otomatis akan dimunculkan sesuai dengan data kendaraan yang kita inputkan pada Form Kendaraan.

    Area yang Berwarna Merah merupakan tabel hasil penyimpanan data transaksi dari area yang Berwarna Biru

    Area yang Berwarna Kuning merupakan area tombol yang memiliki fungsinya masing-masing, untuk tombol :
    - Simpan Transaksi digunakan untuk menyimpan transaksi dari area Berwarna Biru
    - Form User digunakan untuk memanggil Form User
    - Form Kendaraan digunakan untuk memanggil Form Kendaraan
    - Form Pelanggan digunakan untuk memanggil Form Pelanggan
    - Form Keluar digunakan untuk Keluar dari Aplikasi

  • Jika kita klik Tombol Form User akan muncul tampilan seperti gambar di bawah ini:

    Kita bisa menambahkan Username baru agar dapat mengakses aplikasi ini melalui Form Login, dan untuk Ubah, Hapus, dan Non Aktifkan Username kita harus klik terlebih dahulu username yang ada pada tabel, seperti gambar dibawah ini :

  • Jika kita klik Tombol Form Pelanggan akan muncul tampilan seperti gambar di bawah ini: Kita akan coba menginput data, sebagai contoh kita akan menginput data seperti dibawah ini :
    Data yang pertama :
    Nama Pelanggan : Imam Firmansyah
    Alamat : Depok
    No. Telepon : 085781200013
    No. Identitas : 123456789
    Data yang kedua :
    Nama Pelanggan : Dian Sapta
    Alamat : Jakarta
    No. Telepon : 085716607939
    No. Identitas : 987654321
    Dan di bawah ini adalah hasil dari data yang telah kita inputkan :

  • Kita akan coba menginput Data Kendaraan, klik Tombol Form Kendaraan maka akan muncul tampilan seperti di bawah ini : Kita akan coba menginput data, sebagai contoh kita akan menginput data seperti dibawah ini :
    Data yang pertama :
    No. Polisi : B 1234 XYZ
    Nama Kendaraan : Honda - Jazz
    Tahun Kendaraan : 2014
    Biaya Sewa : 250000
    Data yang kedua :
    No. Polisi : B 5678 VWX
    Nama Kendaraan : Toyota - Avanza
    Tahun Kendaraan : 2015
    Biaya Sewa : 200000
    Dan di bawah ini adalah hasil dari data yang telah kita inputkan :

  • Klik Tombol Form Sewa maka akan kembali ke Form Sewa, kita bisa mulai menginput data penyewaan kendaraan, klik Combo Box ID Pelangan data yang telah kita masukan sebelumnya sudah dapat dipanggil melalui combo box, Pilih ID Pelanggan maka akan muncul data pelanggan seperti gambar dibawah ini : Begitu juga dengan data Kendaraan,

  • Kita akan mencoba menginput data transaksi penyewaan, pilih tanggal awal penyewaan dan tanggal akhir penyewaan, sebagai contoh kita inputkan data seperti gambar di bawah ini : Maka pada tabel transaksi akan muncul data yang telah kita inputkan, Klik transaksi pada tabel transaksi untuk melihat detail dari transaksi penyewaan yang telah kita inputkan Terdapat informasi mengenai transaksi penyewaan kendaraan, diantaranya lamanya waktu, biaya sewa dan kendaraan apa yang di sewa serta identitas pelanggan yang menyewa kendaraan tersebut, fungsi tombol Sewa Selesai merupakan tombol yang berfungsi untuk mengganti status dari Di Sewakan menjadi Selesai

  • Karena Data Pelanggan, Data Kendaraan memiliki relasi dengan Data Transaksi Sewa maka setiap kendaraan yang telah disewa, status kendaraan akan berganti dari tersedia menjadi tidak tersedia
